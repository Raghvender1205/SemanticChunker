# Semantic Chunker
A python library to semantically chunk (`code` or `text`) for RAG systems.

## Install
```
cd semantic_chunker
pip install .
```

## Usage
Add Gemini API key
```
export GEMINI_API_KEY="YOUR_GEMINI_API_KEY"
```
